# Code_For_2019_Winter_Camp
***
### 概述

这是广州大学机械与电气工程学院“冬令营”电类课程示例目录，由创客协会提供支持。

***

- [x] AVR入门-定时器/计数器（数码秒表）
- [x] AVR入门-中断（抢答器设计）
- [x] AVR入门-ADC（光控灯制作）
- [x] C语言编程范式
- [x] AVR入门-串口通信
