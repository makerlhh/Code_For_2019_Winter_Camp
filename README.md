# Code_For_2019_Winter_Camp
***
### 概述

这是广州大学机械与电气工程学院“冬令营”电类课程示例目录，由创客协会提供支持。

***
### 电类课程列表
- [x] AVR入门-I/O的使用上（开发与仿真环境）
- [x] AVR入门-I/O的使用下（流水灯与数码管）
- [x] AVR入门-定时器/计数器（数码秒表）
- [ ] AVR入门-中断（抢答器设计）
- [ ] AVR入门-ADC（光控灯制作）
- [ ] C语言编程范式
- [ ] AVR入门-串口通信
- [ ] AVR入门-SPI通信
- [ ] AVR入门-TWI通信
